# Product Blueprint

## Vision
Build an open, Matrix-native collaboration platform where humans and AI agents coordinate work in shared rooms with policy and audit controls.

## Persona Highlights
- Engineering manager
- Platform/SRE lead
- Compliance and security lead
- Product manager
- Enterprise admin

## MVP Scope (90 Days)
Priority 0:
- Matrix room integration and `@agent` invocation
- Sequential multi-agent orchestration with retries and timeouts
- Room-level controls and action timeline
- Seat accounting hooks

Priority 1:
- OIDC-based SSO baseline
- Policy allowlist for sensitive tools
- Usage dashboard for seats and action volume

## Editions
Community:
- Core runtime and APIs
- Basic orchestration
- Self-host docs and community support

Enterprise:
- SSO, SCIM-ready identity integrations
- Advanced RBAC/policy controls
- Compliance exports and stronger tenancy isolation
- Seat license governance and commercial support

## Business Model
- Managed cloud subscriptions
- On-prem annual seat-based licensing
- Open core growth motion for adoption
